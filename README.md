# TruthHubBD

TruthHubBD is a community-driven review platform where users can share honest reviews and ratings about different services and organizations in Bangladesh.

## Features

- ⭐ Review & Rating System
- 🔍 Smart Search
- 👤 User Authentication
- 🏥 Multiple Categories (Doctors, Hospitals, Restaurants, Universities, Products, etc.)
- 📝 User Profiles

## Tech Stack

- **Frontend:** HTML, CSS, Bootstrap, TypeScript
- **Backend:** PHP
- **Database:** MySQL

## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/faysaliqbal007/TruthHubBD.git
   ```

2. Import the database into MySQL.

3. Configure your database connection.

4. Run the project using XAMPP or any PHP server.
